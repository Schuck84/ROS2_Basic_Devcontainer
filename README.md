# ROS2 Basic Devcontainer workspace
This repository contains a basic workspace configuration for development of ROS2 apps inside a devcontainer

## Based on ROS 2 examples
To see some of these examples in use, visit the [ROS 2 Tutorials page](https://docs.ros.org/en/jazzy/Tutorials/Beginner-Client-Libraries/Writing-A-Simple-Cpp-Publisher-And-Subscriber.html).
